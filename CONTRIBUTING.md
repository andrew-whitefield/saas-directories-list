# Contributing

Thanks for helping build the most complete SaaS directory list on the internet.

## Add a directory

You can contribute two ways.

**Quick way (Issue):** open a new Issue using the "Add a directory" form and fill in the name, link, and a short description. We will add it.

**Direct way (Pull Request):**

1. Click the pencil icon on `README.md` to edit it.
2. Add your entry in the correct letter section, keeping the list alphabetical.
3. Use this exact format:
   ```
   - [Name](https://example.com/) - A few words on what it does.
   ```
4. Commit and open a Pull Request.

## Rules

- One entry per directory. Check that it is not already listed (search the page with Ctrl/Cmd+F before adding).
- Keep descriptions to a few words. State what it does, not marketing copy.
- No em dashes. Use a normal hyphen as the separator and commas or periods inside the description.
- Use the real homepage URL with `https://`. No affiliate or referral links.
- Keep entries alphabetical within each letter. Number-led names go in the `0-9` section.
- Only genuine product, SaaS, startup, or AI directories, launch platforms, and review sites. No personal blogs, social media links, or paywalled press pages.
- Confirm the link works and the site is live before submitting.

## What we remove

Dead links, duplicates, spam, and listings that are not actually directories. If a site shuts down, open an Issue and we will remove it.
